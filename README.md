# API-REST-con-JWT-y-Swagger
# 🚀 API RESTful - Proyecto Final

Este repositorio contiene el código fuente de una API REST segura y documentada, desarrollada como proyecto para la clase de la **Prof. Dina**. 

El proyecto implementa autenticación mediante **JSON Web Tokens (JWT)**, documentación interactiva con **Swagger**, y un script de inicialización de base de datos que incluye la estructura y datos semilla (seed).

---

## 🛠️ Tecnologías Utilizadas

* **Entorno/Lenguaje:** [Ej: Node.js con Express / Python con FastAPI / Java con Spring Boot]
* **Base de Datos:** [Ej: PostgreSQL / MySQL]
* **Seguridad:** Autenticación y Autorización con **JWT**.
* **Documentación:** **Swagger / OpenAPI**.
